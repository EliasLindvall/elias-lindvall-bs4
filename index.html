Meta

The meta tag is used for the metadata of the webpage. The meta tag is within the head tag and doesn't have any visual impact but is very helpful for search engine optimization or SEO for short.

Responsive

Web Design

Responsive Web Design is about your web page automatically adjusting for different screen sizes and viewports. For example if you create your web page on the computer without responsive web desing it may not look very good on the phone.

Http

Http stands for Hypertext Transfer Protocol. Http is the foundation of data communication for the World Wide Web and it is what makes clicking hyperlinks on a website possible.

Https

HTTPS stands for Hyper Text Transfer Protocol Secure and is very similar to http except https is more secure, which means that the data is encrypted before sending.

Compressed Data

compressed data or data compression is when data is coded using fewer bits than the original representation. This is helpful in many ways, it can be used to save storage on disc.

CDN

CDN stands for Content Delivery Network and is a geographically distributed network of proxy servers and their data centers which work together for fast delivery of internet content. Using a CDN on your website can boost the speed.

Local Boostrap

Local bootsrap is when you have downloaded the bootstrap file and added it into your project folder where you wanna use it

JQuery

JQuery is a JavaScript library, the purpose of it is to make it much easier to use Javascript in your website.

Flexbox

Flexbox is a one-dimensional layout method for laying out items in rows or columns. Items flex to fill additional space and shrink to fit into smaller spaces.

Grid

The CSS Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning.

Code Snippets

Code snippets are templates that make it easier to enter repeating code patterns, such as loops or conditional-statements.
